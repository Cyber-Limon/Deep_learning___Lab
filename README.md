# Глубинное обучение
# Лабораторная работа
# Математическое и программное обеспечение для прогнозирования свойств медицинских композиционных материалов

Есть два трека выполнения лабораторных работ:
1. В рамках выполнения ВКР лабораторные работы выполняются индивидуально и показываются поэтапно:
- формулировка и формализация задачи, в том числе в терминах ИНС;
- сбор и предобработка данных, формат представления для обучения;
- анализ и обоснование выбора архитектуры и алгоритмов обучения;
- проведение вычислительных экспериментов (разные архитектуры, алгоритмы обучения, оптимизаторы, гиперпараметры, размеры батчей и т.п.) - оформляение таблицы с метриками;
- анализ и интерпретация результатов обучения.
2. Традиционный: задания в файле (можно выполнять в группах по 2-3 человека), в конце оформляется общий отчет по всем лабораторным работам в виде рабочей тетради.
